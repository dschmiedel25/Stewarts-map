How to get a key & what to enable
  1. In Google Cloud → Maps, create a project and an API key.
  2. Enable three APIs: Maps JavaScript API, Places API (New), and Geocoding API.
  3. Restrict the key by HTTP referrer to your site (e.g. yourname.github.io/*) so it can't be abused.
  4. Cost: each location = one Text Search call. The first 5,000 calls per month are free; Waypoint shows the count before you run so you can stay under it.
